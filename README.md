# kintai

```
% npm install --global kintai

% kintai
Usage: kintai [command] [option]
  
Commands:
  start         出勤する
  start [time]  時間を指定して出勤する
  end           退勤する
  end [time]    時間を指定して退勤する
  rest          休憩に入る
  rest [time]   時間を指定して休憩に入る
  return        休憩から戻る
  return [time] 時間を指定して休憩から戻る
  edit          勤怠情報を変更する
  show          勤怠情報を表示する
```
